# Quantum Zombie in a Box

This is a game based on Schrödinger’s cat, but with a zombie. The zombie is in superposition while the box is closed, and when you open it, you randomly get one of four states. There’s a second box to show entanglement, and a “Try to Escape” button that models quantum tunneling.

To play: open `index.html` in your browser and click the buttons.
